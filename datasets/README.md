# Datasets

This directory contains datasets used during the CHESS analysis workshop.

## Contents

Store raw and processed data files here. Suggested organization:

- `raw/` — unprocessed experimental data
- `processed/` — data after reduction or transformation
- `example/` — small example datasets for tutorials

## Notes

Large data files should be tracked with Git LFS or linked externally. Include a brief description of each dataset, its origin, and the conditions under which it was collected.
