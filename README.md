# chess_workshop
Repo for code and documents related to the CHESS analysis workshop held in March 2026.

## Repository Structure

```
chess_workshop/
├── analysis_scripts/   # Scripts for scientific data analysis (reduction, fitting, visualization)
├── datasets/           # Raw and processed experimental datasets
└── documentation/      # Workshop guides, tutorials, and reference documentation
```

Refer to the `README.md` inside each directory for more details on its contents and usage.
